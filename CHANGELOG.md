# Changelog

## 1.1.0 - 2026-07-18

- Added the real Task'sList mark to custom window chrome and embedded the ICO for WPF windows, the executable, taskbar, and notification area.
- Made the complete main-window title strip hit-testable and draggable with a hand cursor while preserving native-feeling double-click maximize/restore.
- Fixed a region-capture crash caused by hiding a modal WPF overlay before assigning `DialogResult`; capture faults are now contained and reported without terminating Task'sList.
- Rebuilt stickies as durable desktop objects with title dragging/editing, snapping, lock, roll, ghost recovery, multi-monitor rescue, 20–100% focus opacity, and complete per-note appearance controls.
- Added tray lifecycle, configurable global hotkeys, application attachment modes, reminders, sleep/wake, 30-day Trash, and recoverable archive behavior.
- Added a ClipAngel-style clipboard palette with provenance, filters, favorites, duplicate promotion, multiselect, paste variants, editing, assignment, and note conversion.
- Added safe portable progress, counter, and timer Markdown controls.
- Replaced placeholder manager content with a polished real-data Notes, Clipboard, Contexts, Extensions, and Trash library.
- Added high-contrast/reduced-motion behavior, semantic automation names, live snap settings, bulk style application, and persistent editor/preview mode.
- Added an installed-artifact probe and strengthened release asset/checksum verification.

## 1.0.0 - 2026-07-18

- Initial Task'sList Windows release with contextual sticky notes, Markdown, themes, clipboard history, Places, browser tabs, region capture, and three showcase plugins.
